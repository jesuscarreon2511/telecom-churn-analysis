# Telecom Churn Analysis

## Descripción del Proyecto

Este proyecto analiza el abandono de clientes (churn) en una empresa ficticia de telecomunicaciones llamada Telecom X.  
El objetivo del análisis es explorar los datos de los clientes para identificar patrones y factores que influyen en la cancelación del servicio.

El análisis incluye procesos de limpieza de datos, exploración del dataset y preparación de variables que posteriormente pueden utilizarse en modelos de Machine Learning para predecir el churn.

---

## Objetivos del Análisis

- Explorar el dataset de clientes de Telecom X.
- Identificar valores nulos o inconsistencias en los datos.
- Analizar la variable objetivo **Churn**.
- Preparar el dataset para futuros modelos de predicción.

---

- Python

## Dataset

El dataset utilizado proviene del repositorio oficial del challenge:

https://raw.githubusercontent.com/ingridcristh/challenge2-data-science-LATAM/main/TelecomX_Data.json

El conjunto de datos contiene información sobre clientes, incluyendo:

- Información del cliente
- Servicios contratados
- Tipo de contrato
- Facturación
- Estado de cancelación del servicio (Churn)

---

## Proceso de Análisis

El flujo del análisis realizado en el notebook incluye:

1. Carga del dataset desde el repositorio remoto.
2. Exploración inicial de los datos.
3. Revisión de valores nulos y duplicados.
4. Limpieza y transformación de variables.
5. Conversión de la variable **Churn** a formato numérico.
6. Preparación del dataset para análisis posteriores.
